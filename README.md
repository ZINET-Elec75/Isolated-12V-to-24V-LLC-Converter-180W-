# Isolated-12V-to-24V-LLC-Converter-180W-PCBsize (65 mm × 25 mm)-
180W isolated DC/DC converter based on LLC resonant topology. 12V to 24V step-up design using a planar transformer, optimized for high efficiency and power density. Designed as an auxiliary power supply for gate drivers and embedded power systems.

## 📖 Overview
This project presents the design of a 180W isolated DC/DC converter based on an LLC resonant topology.

The converter steps up a 12V DC input to a 24V DC output and is intended for auxiliary power supply applications such as gate driver powering in high-power systems.

The design integrates a planar transformer to achieve high efficiency, compact size, and improved thermal performance.

## 🧠 Design Highlights
- LLC resonant topology enabling soft switching (ZVS)
- High efficiency at medium power level (180W)
- Planar transformer for high power density and low profile design
- Open-loop control for simplified implementation
- Optimized resonant tank design (Cr, Lr, Lm)
